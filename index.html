Privacy Policy

LinkedIn Comment Assistant

Last updated: 2025-08-11

This Chrome extension helps you generate short, context-aware comments for LinkedIn posts. We take privacy seriously and designed the extension to minimize data access and avoid developer-side storage.

What data the extension accesses

  • Visible LinkedIn post text on the page where you click “Generate Comment”.
  • User settings you provide in the Options page, such as model, tone, max words, and optional key message.
  • OpenAI API key, if you choose to enter one, stored locally in Chrome sync storage on your device.

How data is used

  • When you click “Generate Comment”, the extension sends the post text and your settings directly from your browser to the OpenAI API at api.openai.com using your own API key.
  • The OpenAI response is rendered immediately on the LinkedIn page for your use. No server controlled by the developer receives any data.

What the developer does not do

  • No collection of personally identifiable information.
  • No developer-operated servers receive, store, or log your content or API key.
  • No sale of data, no sharing with advertisers.

Permissions rationale

  • storage: Save your API key and preferences locally in Chrome sync storage for convenience.
  • activeTab: Run on the current LinkedIn tab when you use the extension.
  • scripting: Inject the “Generate Comment” UI into the LinkedIn page.
  • Host permission (https://api.openai.com/*): Communicate directly with the OpenAI API to generate comments.

Third‑party processing

The OpenAI API processes the text you send to generate a comment. You control this flow via your own API key. Please review OpenAI’s policies and terms before use.

Data retention

The extension does not retain data outside your browser’s local storage. You can remove your API key and settings anytime from the Options page or by uninstalling the extension.

Children’s privacy

This extension is intended for use by adults. It does not knowingly collect information from children.

Changes to this policy

If this policy changes, the “Last updated” date above will be revised. Significant changes may be mentioned in the Chrome Web Store listing.
